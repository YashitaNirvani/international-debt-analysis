# International Debt Analysis

## Project Overview
This project analyzes International Debt Statistics (IDS) published by the World Bank.  
The objective is to examine global and country-level debt trends, measure debt volatility over time, and derive insights using SQL and Python.

---

## Dataset
Source: World Bank – International Debt Statistics

The dataset includes:
- Country-level external debt indicators
- Time-series data spanning multiple years
- Economic and financial debt metrics

---

## Tools and Technologies
- SQL – data cleaning, transformations, joins, and window functions
- Python – data analysis and visualization
  - pandas
  - matplotlib / seaborn
- Relational database for structured storage of IDS tables

---

## Analysis Performed
- Cleaning and standardizing raw IDS data
- Handling missing and null values
- Year-over-year debt change analysis
- Debt volatility calculations using window functions
- Country-wise and region-wise comparisons
- Trend analysis and visualization

---

## Visualizations
- Line charts showing debt trends over time
- Comparative charts across countries
- Volatility patterns by year
